# Crime Hotspot Prediction Using Spatial Machine Learning

## Overview

This project predicts crime hotspots in Chicago using historical crime records and spatial machine learning techniques.

The workflow combines:

- Spatial Data Science
- Feature Engineering
- XGBoost Classification
- GIS Visualization
- Interactive Maps

---

## Dataset

Chicago Crime Dataset (Official Open Data Portal)

Source:
https://data.cityofchicago.org/Public-Safety/Crimes-2001-to-Present

---

## Project Workflow

Historical Crime Data
↓
Latitude & Longitude
↓
Spatial Grid Cells
↓
Crime Counts per Region
↓
Hotspot Label Generation
↓
XGBoost Model
↓
Interactive GIS Map
